# 🌸 Pearl.exe — Interactive Portfolio

An immersive retro-inspired portfolio experience built with pure HTML, CSS, and JavaScript.

Instead of a traditional scrolling portfolio, **Pearl.exe** transforms a developer portfolio into a fully explorable pixel-art world — complete with rooms, NPC interactions, collectibles, achievements, animated UI, and game-inspired navigation.

---

## ✨ Features

- 🎮 Interactive pixel-art portfolio world
- 🏠 Explorable themed rooms
- 🗨️ NPC dialogue system
- 🏆 Achievement & collectible system
- 🗺️ Real-time minimap
- 📱 Mobile touch controls
- 🌸 Animated retro UI
- 💻 Pure HTML, CSS & JavaScript
- ⚡ No frameworks or libraries

## 🚀 Running Locally

Clone the repository:

```bash
git clone https://github.com/Pearl-Omphile/portfolio.git
```

Open the project folder:

```bash
cd portfolio
```

Run the project by opening:

```bash
index.html
```

in your browser.
